# [Dynamic Grade Search Site](https://www.linkedin.com/posts/mohamed-e-mad_reactjs-nodejs-mongodb-activity-7151674362679472128-NwMF?utm_source=share&utm_medium=member_desktop)🚀

Excited to unveil my latest project: a dynamic website using **React**, **Node.js**, **Bootstrap**, and **MongoDB**! 🌟

## Project Overview

The idea behind this project is to simplify the search for high school certificates in Egypt for the year 2023. By converting Ministry of Education-provided XLSX sheets into a database, I've enabled seamless grade searches. 📄🔍

## Features

- 🔎 **Search by Name or ID**: Easily find grades with a hassle-free search experience.
- 🌐 **Deployed API**: The API is live, offering real-time grade search functionality and enhancing accessibility and convenience.

## Tech Stack

- **Frontend**: React and Bootstrap for a responsive and intuitive user interface. ⚛️✨
- **Backend**: Node.js for server-side logic. 🛠️
- **Database**: MongoDB for robust data storage. 📦
- **Deployment**: The API is deployed and accessible for live use. 🚀
